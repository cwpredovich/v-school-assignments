The Almighty Input Tag

Here are some HTML <input> tag examples showing how to use the different input type attributes.
The basic text input: 
                        <input type="text" placeholder="text"/>
The telephone number input: 
                        <type="tel" placeholder="telephone #" />
The email address input: 
                        <input type="email" placeholder="email addy"/>
The date input which has a dropdown calendar: 
                        <input type="date" />
The radio button which allows you to pick one item: 
                        <input type="radio" name=yes-or-no value="yes">Yes?<br>
                        <input type="radio" name=yes-or-no value="no">No?<br>
